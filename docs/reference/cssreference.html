CSS Properties Reference

Interaction between GLAM scene graph and CSS layout

The CSS layout rules for the page stop at the GLAM element. For all descendants of the GLAM element, the CSS properties specify 3D scene graph units, not page units. With the exception of 3D transforms, standard CSS layout properties do not apply.

CSS Transform properties

GLAM uses CSS transforms to place objects in the 3D scene.

CSS Animation and Transition properties

GLAM uses CSS Animations and CSS Transitions to dynamically animate or transition property values.

Visual properties: 'dash-size', 'gap-size', 'line-width', 'vertex-colors', and 'vertex-normals'

Visual properties specify values for various visual elements such the width of lines, and how colors and normals are organized for meshes, points and lines.

'depth'
Value:  <number>
Initial:  1
Applies to:  visual elements

This property specifies the depth (z axis extent) of visual elements, such as the depth of a cube, in model space.

'radius'
Value:  <number>
Initial:  1
Applies to:  visual elements

This property specifies the radius of visual elements, such as a sphere, in model space.

'radius-segments'
Value:  <number>
Initial:  16
Applies to:  visual elements

This property specifies how many vertical segments will be used to tesselate primitive shapes such as cones and cylinders.

'dash-size'
Value:  <number>
Initial:  0
Applies to:  visual elements

This property specifies the dash size for dashed lines. A value of 0 means no dash.

'gap-size'
Value:  <number>
Initial:  0
Applies to:  visual elements

This property specifies the gap size for dashed lines. A value of 0 means no gap.

'line-width'
Value:  <number>
Initial:  auto
Applies to:  visual elements

This property specifies the width (thickness) of lines. The default is implementation-specific.

'vertex-colors'
Value:  <vertex-colors>
Initial:  auto
Applies to:  visual elements

This property, if present, specifies that colors within a mesh are supplied one per vertex rather than one per face.

'vertex-normals'
Value:  <vertex-normals>
Initial:  auto
Applies to:  visual elements

This property, if present, specifies that normals within a mesh are supplied one per vertex rather than one per face.

3D Text properties: 'bevel-size', 'bevel-thickness', 'font-bevel', and 'font-depth'

Properties to add to CSS for 3D text, including bevel and thickness of the extruded text.

'bevel-size'
Value:  <number>
Initial:  auto
Applies to:  text elements

This property specifies the bevel size for beveled 3D text.

'bevel-thickness'
Value:  <number>
Initial:  auto
Applies to:  text elements

This property specifies the bevel thickness for beveled 3D text.

'font-bevel'
Value:  bevel | none
Initial:  bevel
Applies to:  text elements

Specifies whether to bevel 3D text.

'font-depth'
Value:  <number>
Initial:  0
Applies to:  text elements

Specifies the depth to extrude 3D text along the model space z axis. A value of 0 means the text is not extruded but rendered as flat polygons.
