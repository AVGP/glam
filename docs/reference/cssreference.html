CSS Properties Reference

Interaction between GLAM scene graph and CSS layout

The CSS layout rules for the page stop at the GLAM element. For all descendants of the GLAM element, the CSS properties specify 3D scene graph units, not page units. With the exception of 3D transforms, standard CSS layout properties do not apply.

CSS Transform properties

GLAM uses CSS 3D transforms to place objects in the 3D scene by setting the transform property, such as in the following example:

#photocube { image:url(../../images/flowers.jpg); transform:rotateX(30deg); }

Elements inherit the 3D transforms of the their ancestors.

CSS Animation and Transition properties

GLAM uses CSS Animations and CSS Transitions to dynamically animate or transition property values.

The following CSS fragment defines a keyframe animation to rotate the cube cube1 about its y-axis over a 10-second duration with looping:

/* keyframe animation for the cube: 10-second looping rotation about y-axis */ @keyframes kfRotateY { from { transform: rotateY(0deg); } to { transform: rotateY(360deg); } } #cube1 { animation-duration: 10s; animation-name: kfRotateY; animation-iteration-count: infinite; animation-timing-function:linear; }

Material properties

Visual properties

The visual properties 'dash-size', 'gap-size', 'line-width', and 'point-size' specify values for various visual elements such the width of lines and size of points.

'dash-size'
Value:  <number>
Initial:  0
Applies to:  visual elements

This property specifies the dash size for dashed lines (<lines> element only). A value of 0 means no dash.

'gap-size'
Value:  <number>
Initial:  0
Applies to:  visual elements

This property specifies the gap size for dashed lines (<lines> element only). A value of 0 means no gap.

'line-width'
Value:  <number>
Initial:  auto
Applies to:  visual elements

This property specifies the width (thickness) of lines (<lines> element only). The default is implementation-specific.

'point-size'
Value:  <number>
Initial:  0
Applies to:  visual elements

This property specifies the point size for points (<points> element only). A value of 0 means use a default size.

3D Text properties

Properties to add to CSS for 3D text, including 'bevel-size', 'bevel-thickness', 'font-bevel', and 'font-depth'.

'bevel-size'
Value:  <number>
Initial:  auto
Applies to:  text elements

This property specifies the bevel size for beveled 3D text.

'bevel-thickness'
Value:  <number>
Initial:  auto
Applies to:  text elements

This property specifies the bevel thickness for beveled 3D text.

'font-bevel'
Value:  bevel | none
Initial:  bevel
Applies to:  text elements

Specifies whether to bevel 3D text.

'font-depth'
Value:  <number>
Initial:  0
Applies to:  text elements

Specifies the depth to extrude 3D text along the model space z axis. A value of 0 means the text is not extruded but rendered as flat polygons.
